# Chaeryeong Diary Website This repository hosts the official fan page for **Chaeryeong from ITZY** — a space to celebrate her performances, beauty, and artistry. ## Files - `index.html` — Main page of the Chaeryeong Diary fan site. - `.nojekyll` — Prevents GitHub Pages from ignoring files starting with underscores. - `README.md` — Overview and deployment instructions. ## How to Publish on GitHub Pages 1. Create a new GitHub repository named **chaeryeongdiary**. 2. Upload all files (including `index.html`, `.nojekyll`, and `README.md`). 3. Go to **Settings → Pages**. 4. Under **Branch**, select `main` (or `master`) and click **Save**. 5. After a few minutes, your website will be live at: ``` https://.github.io/chaeryeongdiary ``` 6. To use your custom domain (**chaeryeongdiary.com**): - Add a **CNAME** file containing: ``` chaeryeongdiary.com ``` - Go to your domain registrar and add a CNAME record pointing to `yourusername.github.io`. ✨ _Made with love by MIDZYs for Chaeryeong._
